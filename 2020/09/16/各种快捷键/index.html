window10系统中各种快捷键

win+任意键篇

  • win + Shift + C 可以呼叫我们的小娜

  • win + A 打开右下角的操作中心

  • win + R 打开运行窗口

  • win + D 立即来到桌面按下第二次可以快速恢复之前的模样

  • win + G 可以打开游戏录制

  • win + L 立即锁定桌面

  • win + M 跟 Win + D 一样快速来到桌面，但是重复按下不能恢复（对于老板来检查或者老师检查很有用😁😁）

  • Win + Tab 相当于手机的最左边那个键，可以看到正在运行的所有应用，可以快速切换界面

  • Win + W 打开Window Ink工作区 工作区还是很不错的（电脑如果有接触屏或者接触设备之类的会更好用）

  • Win + X 立即打开【开始】的右键菜单栏

  • Win + U 立即打开【设置】

  • Win + Q 打开搜索栏扩展

  • Win + P 设置第二屏幕

  • Win + K 选择无线设备

  • Win + V 打开剪切版详细界面

  • Win + 》打开表情包（当时我发现的时候我都惊了🤣🤣）

好了，就这么多了，以后发现了再补充说明，就这样，感谢坚持到这里的朋友。

  • ctrl + z 撤回
  • ctrl + x 剪切
  • ctrl + c 复制
  • ctrl + v 粘贴
  • ctrl + F4 关闭当前窗口
  • ctrl + W或ctrl + F4关闭浏览器当前页面
